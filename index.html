Lego(IoT) House

Featured Project

This Lego(IoT) house may look like a toy, but don't be fooled! This Lego(IoT) House uses an Arduino board to control and receive data through the browser from numerous sensors, lights and motors integrated into the house. I used Node.js as my server and both web sockets and PubNub's network to send real-time messages globally. The project can stream data from the house to plot.ly, an online analytics and data visualization tool, where it can be referenced in real time. This data, and the unique ways it can be visualized, can help find and solve problems and issues that we might not recognize as being problems yet.
